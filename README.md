# Reproducible Data Analysis

This repository contains my work for HW #4.3 and HW #4.4, focusing on data visualization and Monte Carlo numerical integration using Quarto.

## Overview
The project analyzes airport passenger traffic trends and uses Monte Carlo simulation to estimate the area under a curve. It demonstrates reproducible workflows using Quarto and R.

## Interesting Insight
The Monte Carlo simulation shows that as the number of points increases, the estimate becomes more accurate and converges near the true value (~0.78).

- Airport dataset provided in class
- Simulated data generated in R
